# Node.js + Express.js Revision Repository

This repository is dedicated to revising and mastering **Node.js** and **Express.js** — from core backend fundamentals to building real-world REST APIs.  
It includes concepts, mini projects, and daily practice files.

---

## Topics Covered

### **Node.js Fundamentals**
- What is Node.js?
- Modules (`require`, `exports`)
- `os` module
- `fs` module (File System)
- Events & EventEmitter
- Streams and Buffers

### **Express.js Basics**
- What is Express and why it’s used
- Setting up an Express server
- Handling routes (`GET`, `POST`, `PUT`, `DELETE`)
- Working with Middleware
- Serving static files
- Express Router for modular apps

### **Advanced Topics**
- RESTful API design
- Error handling middleware
- Environment variables (`dotenv`)
- Working with JSON data
- Async/Await with APIs
- Connecting Express with MongoDB (optional)

---

## Setup Instructions

1. **Initialize Project**
     ```
     npm init -y
     ```
2. **Install Dependencies**
    ```
    npm install express nodemon
    ```
3. **Run Server**
   ```
   node app.js
   ```
4. **or (for live reload)**
   ```
   npx nodemon app.js
   ```
